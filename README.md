# Hidden Discount

Hidden Discount is a small Expo + React Native app that demonstrates an interactive shopping card flow:
- Swipe the product card to reveal a hidden Black Friday discount code.
- Copy the code and apply it at checkout.
- Update quantity and see pricing reflect the applied discount.

## Core Tech
- Expo (SDK 55) + React Native + TypeScript
- Expo Router for app structure/navigation
- Uniwind + Tailwind CSS v4 for styling
- React Native Reanimated + Gesture Handler for swipe interactions
- Bun for package management and scripts
